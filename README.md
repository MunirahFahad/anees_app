# أنيس (Anees)

## الوصف
"أنيس" هو تطبيق يعتمد على الذكاء الاصطناعي لتحويل المواقع التراثية إلى تجربة صوتية تفاعلية، تهدف إلى تمكين ذوي الإعاقة البصرية من استكشاف الأماكن بشكل مستقل من خلال السرد الحسي والتفاعل الصوتي.

## المشكلة
تعتمد أغلب التجارب السياحية على الرؤية، مما يحد من قدرة ذوي الإعاقة البصرية على استكشاف المواقع التاريخية بشكل مستقل.

## الحل
يوفّر "أنيس" تجربة إرشادية صوتية تفاعلية تجمع بين:
- وصف حسي للمكان
- شرح تاريخي مبسط
- إمكانية التفاعل الصوتي وطرح الأسئلة

## التقنيات المستخدمة
- Flutter (Dart)
- REST APIs
- nuha-llm
- elm-tts
- elm-asr
